Master HTML 5 WP

Integración Masiva de proyectos open Source para generar un tema starter de WP.

Enfocado en Diseñadores y programadores que inician con Wordpress

Salidas de LESS:

Skin que se monta sobre Bootstrap
Origen:			/library/less/style.less
Destino:		/library/css/style.css

Version IE del skin que se monta sobre Bootstrap
Origen:			/library/less/ie.less
Destino:		/library/css/ie.css


Skin para el sistema de acceso de WP
Origen:			/library/less/login.less
Destino:		/library/css/login.css


LESS de bootstrap 

Bootstrap
Origen:			/library/less/libs/bootstrap-3.1.1/bootstrap.less
Destino:		/library/css/libs/bootstrap-3.1.1/bootstrap.css

Bootstrap theme (Efectos de volumen sobre los assets de bootstrap aka. que el botón se vea pachón)
Origen:			/library/less/libs/bootstrap-3.1.1/theme.less
Destino:		/library/css/libs/bootstrap-3.1.1/bootstrap-theme.css ¡¡Atención al cambio de nombre!!


CONFIGURACIONES DE Bootstrap:

Utilizar archivo de "/library/less/libs/bootstrap-3.1.1/variables.less"  para cambiar los colores, typos, etc  que bootstrap trae por default.
